Cómo ejecutar  Siaco Infinity Tech

1. Descargue el archivo zip

2. Extraiga el archivo y copie la carpeta

3.Pegue dentro del directorio raíz (para xampp xampp/htdocs, para wamp wamp/www, para lamp var/www/html)

4. Abra PHPMyAdmin (http://localhost/phpmyadmin)

5. Importe el archivo hms.sql (que se encuentra dentro del paquete zip en la carpeta de archivos SQL)

6.Ejecute el script http://localhost/citas/ (frontend)
